# План: UI Testing (Web)

**Цель**:  
Проверить работоспособность, безопасность и стабильность Web-версии мессенджера на Flutter.

**Scope**:  
- Вход, создание чата, отправка сообщений (Smoke),  
- Обработка XSS, Unicode, эмодзи, длинных сообщений,  
- Поведение при недоступности бэкенда,  
- Функционал WebSocket (уведомления),  
- Отображение кириллицы и спецсимволов.

**Инструменты**:  
- `flutter run -d chrome/edge`,  
- DevTools (Console, Network),  
- `curl` для подготовки данных,  
- `adb logcat` — не применялся (Web),  
- MySQL CLI для проверки состояния БД.

**Критерии успеха**:  
✅ Все основные сценарии работают,  
✅ XSS не исполняется,  
✅ UI не крашится при граничных/некорректных данных,  
✅ При потере связи — понятное поведение.

**Выход**:  
- `ui_web_test_report.md` — полный отчёт.
